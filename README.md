# ⚠️ SecurePay (Vulnerable Version)

An intentionally vulnerable online payment application developed for educational purposes to demonstrate common web application security vulnerabilities.

> **Disclaimer:** This project is intended strictly for learning and ethical cybersecurity education. Do not deploy it in production environments.

---

## 📌 Overview

SecurePay Vulnerable is a Node.js web application designed to help students understand how common web application vulnerabilities can be exploited and analyzed in a controlled environment.

The project was developed as part of a Network Security course.

---

## ✨ Features

- User Authentication
- Payment Processing
- Customer Dashboard
- MongoDB Database
- REST APIs

---

## 🔓 Demonstrated Vulnerabilities

- Cross-Site Scripting (XSS)
- Injection Attacks
- Weak Authentication
- Missing HTTP Security Headers
- Information Disclosure
- Insecure Logging
- Weak Input Validation
